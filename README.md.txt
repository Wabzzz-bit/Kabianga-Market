## ☁️ Hosting for Free

### 🟢 GitHub Pages

1. Go to your repository → **Settings → Pages**
2. Under “Branch,” select `main` and click **Save**
3. Wait a few minutes, then visit:
   👉 `https://your-username.github.io/kabianga-market`

### 🔵 Netlify (Alternative)

1. Go to [https://www.netlify.com/](https://www.netlify.com/)
2. Click **Deploy site**
3. Drag and drop your project folder
4. Netlify will give you a live URL instantly
